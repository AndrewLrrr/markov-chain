
# Простая реализация генерации текстов на основе цепи Маркова

## В качестве примера был взят список русских пословиц из викицитатника

### Результаты первой генерации 20 текстов:

```
Не в рот тесен.
Не по полету.
Не всё золото, что в рот не без хлеба не в окошко.
Будет суд — не без ума не в лес смотрит.
Не по вкусу, а в том же беда.
Без привету нет — не жить.
Не всё не без головы; без хлеба не без хлеба не без ума не без хозяйки изба новая, а в колокол, а по заслугам его малюют.
Не по милу хорош.
Не все правят.
На что в сорок не без хлеба не без покаяния — и в том и не без глазу.
Не учи безделью, а в рот положить.
Не говоря худого слова сказать.
Не всё не в рот не верит.
Без ума не дадут — и не будешь.
Не по чём она сторожит.
Бог даст.
У кого много причин, тот и в неё попадешь.
В гостях что в будни (то есть — не будешь.
Не всё то и в рот не без алтына горюют.
Жид как сорока не без разума беда.
```

### Результаты второй генерации 20 текстов:

```
В гостях хорошо, а в праздник хуже незнания.
Не по силам, тянись по дрова.
Беда да в боку не поешь.
Не тот и в поле не без хлеба — не будешь.
Не говоря худого слова скажи, дурака — не без героя.
Не по лесу мало?
Без хлеба не пропей ума.
Не по речам.
Не по силам, тянись по милу хорош.
Не по ухо — не знает.
Без хлеба не будет.
Не всё не будешь.
Не много затей.
Без ума голова в праздник и на ласковое не без доли: то не мечи в лес пошёл, тот и в рот да в честь, а на медведя ходят.
Была бы в лес не хочется, а в калашный ряд.
Не ищи кротости!
Не всё то смешно, что за глаза - как свинья.
Ни радости вечной, ни будь урод, а не будешь.
С постылым и в лес не без хлеба жевать.
Не та и виноватый прав.

```
